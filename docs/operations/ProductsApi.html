This is the API

productsGet

The Products endpoint returns information about the *Uber* products\noffered at a given location. The response includes the display name\nand other details about each product, and lists the products in the\nproper display order.\n

URL

https://api.uber.com/v1/products

HTTP Method

GET

Response Type

List

Parameters

    Latitude component of location.

    Longitude component of location.
